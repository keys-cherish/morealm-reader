## 书架

书架是 App 的主页面，管理所有已导入的书籍。

### 视图模式

- **网格模式**（默认）：以封面缩略图展示，适合浏览
- **列表模式**：紧凑排列，显示更多信息

点击右上角图标切换视图模式。

### 文件夹分组

- 长按书籍可进入批量管理模式
- 批量选中后可移入文件夹、置顶、删除
- 点击文件夹进入，返回键退出

### 书架搜索

点击搜索图标，输入书名快速定位书架中的书籍。

### 日夜模式切换

点击书架顶栏的日/夜图标，一键切换日间/夜间主题。

## 导入书籍

### 支持格式

| 格式 | 说明 |
|------|------|
| EPUB | 主流电子书格式，支持封面和目录提取 |
| TXT | 纯文本，自动识别章节目录 |
| PDF | 逐页渲染 |
| MOBI / AZW3 | Kindle 格式 |
| CBZ / CBR | 漫画格式 |

### 导入方式

1. **文件管理器**：用系统文件管理器打开电子书文件，选择「墨境」打开
2. **书架导入**：点击书架右上角「+」按钮，从本地文件选择导入
3. **书源搜索**：在「发现」页搜索在线书源内容，点击即可加入书架

### 两阶段导入

导入大文件时，墨境采用两阶段策略：
1. 先快速提取封面和元数据，书籍立即出现在书架
2. 后台异步解析完整内容（章节、目录等）

## 阅读器

点击书架中的书籍即可进入阅读器。

### 基本操作

- **翻页**：左右滑动 / 点击屏幕左右区域 / 音量键（需在设置中开启）
- **呼出菜单**：点击屏幕中央区域
- **返回**：点击左上角返回按钮或系统返回手势

### 顶部菜单

- **目录**：查看和跳转章节
- **书签**：添加/管理书签
- **全文搜索**：搜索当前书籍内容
- **导出 TXT**：将当前书籍导出为纯文本

### 底部菜单

- **进度条**：拖动快速跳转
- **上一章 / 下一章**：章节切换
- **自动翻页**：开启后自动翻页，可调节速度
- **样式设置**：打开阅读样式面板

### 阅读样式面板

在底部菜单点击样式按钮，可调节：

#### 阅读预设

5 套内置预设，每套自动适配日间/夜间配色：

| 预设 | 日间 | 夜间 |
|------|------|------|
| 纸质 | 暖白底 + 深灰字 | 深黑底 + 灰白字 |
| 护眼 | 浅绿底 + 深绿字 | 深绿底 + 浅绿字 |
| 海蓝 | 浅蓝底 + 深蓝字 | 深蓝底 + 浅蓝字 |
| 暖黄 | 浅黄底 + 棕色字 | 深棕底 + 暖黄字 |
| 墨白 | 纯白底 + 纯黑字 | 纯黑底 + 灰白字 |

#### 亮度

- 拖动滑块调节阅读器亮度（独立于系统亮度）
- 勾选「跟随系统」恢复系统亮度

#### 字号与字体

- 拖动滑块调节字号（支持范围较大）
- 内置多款字体：思源宋体、系统默认等
- 支持导入自定义字体文件（TTF/OTF）

#### 行距与段距

- 行距：紧凑 / 适中 / 宽松 多档可选
- 段间距：多档可选

#### 页边距

- 分别调节左右、上、下页边距

#### 翻页动画

| 动画 | 说明 |
|------|------|
| 平移 | 左右平移切页 |
| 覆盖 | 新页覆盖旧页 |
| 仿真 | 贝塞尔曲线模拟真实翻书效果 |
| 上下滚动 | 连续滚动模式 |
| 无动画 | 直接切换 |

#### 其他

- **繁简转换**：简体 / 繁体 / 关闭
- **屏幕方向**：自动 / 竖屏 / 横屏
- **文字可选择**：开启后可长按选中文字
- **背景图片**：自定义阅读背景图
- **自定义 CSS**：高级用户可注入自定义样式

## 阅读设置

在「我的」→「阅读设置」中配置全局阅读行为：

- **音量键翻页**：使用音量键翻页
- **启动后继续上次阅读**：打开 App 自动跳转到上次阅读位置
- **长按文字划线**：长按文字添加下划线标注
- **显示系统状态**：阅读时显示状态栏
- **显示章节名**：页面顶部显示当前章节
- **显示时间电量**：页面底部显示时间和电量
- **自定义章节正则**：自定义 TXT 章节识别规则（留空使用内置规则）
- **跳过内容正则**：朗读时跳过匹配的段落（如作者注、广告）
- **屏幕关闭时间**：阅读时的屏幕常亮时间

## 发现（书源搜索）

「发现」页用于通过书源在线搜索书籍。

### 使用方法

1. 先在「我的」→「书源管理」中导入书源
2. 切换到「发现」页，输入书名或作者名
3. 点击搜索，等待各书源返回结果
4. 点击搜索结果即可开始阅读，书籍自动加入书架

### 书源管理

在「我的」→「书源管理」中：

- **导入书源**：支持 URL 订阅和 JSON 文件导入
- **兼容 Legado**：直接导入 Legado 格式的书源 JSON
- **启用/禁用**：控制哪些书源参与搜索
- **删除**：移除不需要的书源

### 规则引擎

书源支持四种解析模式：
- **CSS 选择器**：适合结构化 HTML
- **XPath**：适合复杂 HTML 结构
- **JSONPath**：适合 JSON API
- **正则表达式**：适合非标准格式

## 听书（TTS 朗读）

在阅读器中或「听书」页使用 TTS 朗读功能。

### TTS 引擎

| 引擎 | 说明 |
|------|------|
| Edge TTS | 微软神经语音，音质好，需联网 |
| 系统 TTS | 使用系统自带 TTS 引擎，离线可用 |
| OpenAI | OpenAI TTS API |
| 自定义 API | 接入自定义 HTTP TTS 服务 |

### 控制

- **语速调节**：拖动滑块调节朗读速度
- **通知栏控制**：朗读时通知栏显示播放控制
- **音量键切换**：音量键切换上/下段落

## 主题切换

在「我的」页面的「主题切换」区域管理主题。

### 6 套内置主题

| 主题 | 风格 |
|------|------|
| 墨境 | 暗紫色玻璃拟态，默认主题 |
| 纸上 | 暖白纸质感，经典日间主题 |
| 赛博朋克 | 霓虹粉色，科幻风格 |
| 森林 | 深绿色，护眼暗色主题 |
| 深夜 | 纯黑 AMOLED，省电 |
| 墨水屏 | 高对比灰度，适配墨水屏设备 |

### 自定义主题

点击「自定义主题」进入主题编辑器，可自定义：
- 主色调、强调色
- 背景色、表面色、文字色
- 阅读器背景色和文字色

### 主题导入导出

- **导出**：将当前主题导出为 JSON 文件
- **导入**：导入 JSON 主题文件或 Legado 主题格式
- **长按删除**：长按自定义主题可删除

## 备份与同步

### WebDAV 云同步

在「我的」→「WebDAV 同步」中配置：

1. 填写 WebDAV 服务器地址、用户名、密码
2. 支持同步内容：进度 / 书架 / 书源 / 主题

### 本地备份

- **导出备份**：将书架、进度、书源、书签导出为 ZIP 文件
- **导入备份**：从 ZIP 文件恢复数据

---

## 其他设置

### 正文替换净化

在「我的」→「正文替换净化」中配置：
- 添加替换规则，去除广告、净化正文内容
- 支持正则表达式替换

### 离线缓存

在「我的」→「离线缓存」中：
- 批量下载章节，支持离线阅读

### 阅读统计

在「我的」页面顶部查看：
- 总阅读书籍数
- 总阅读时长
- 连续阅读天数
- 今日阅读时长
- 年度阅读报告（可保存为图片分享）

### 应用日志

在「我的」→「应用日志」中查看运行日志和错误信息，便于排查问题。

### Legado 一键搬家

支持导入 Legado 备份数据，书源、书架、进度全部迁移。
